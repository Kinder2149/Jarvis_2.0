Configuration de JARVIS. Les clés API ne sont jamais transmises à l'extérieur.

Clés API

Configurez vos clés API pour accéder aux modèles IA. Les clés sont stockées localement en SQLite.

ℹ️ Seule la clé OpenRouter est nécessaire. Les autres clés sont optionnelles : Anthropic et Google permettent d'appeler ces modèles directement (sans marge OpenRouter). Brave Search active la recherche web dans le Chat.
OpenRouter Requis Accès à tous les modèles : Claude, Gemini, Llama...
Anthropic Optionnel Accès direct à Claude (sans frais OpenRouter)
Google AI Optionnel Accès direct à Gemini (sans frais OpenRouter)
Brave Search API Optionnel Active la recherche web dans le module Chat

Modèles par type de tâche

JARVIS sélectionne automatiquement le modèle optimal selon le type de tâche.

🧭 Routage / Décision Choix du prochain step, décisions rapides, navigation workflow
📊 Structuration Extraction et structuration de données, parsing JSON
💻 Code / Analyse Génération de code, analyse de fichiers, refactoring
🧠 Missions complexes Raisonnement long, architecture, missions critiques
